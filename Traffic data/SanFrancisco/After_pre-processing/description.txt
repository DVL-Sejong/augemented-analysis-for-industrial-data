SanFrancisco_traffic_event.csv
---------------------------------------------------------------------------------------------------------------------------------
Type : 사건의 유형 ex) 사고, 혼잡
Severity : 사고의 심각도 0~4 / 0은 traffic에 미치는 영향이 가장 적음(짧은 지연) / 4는 트래픽에 상당한 영향(긴 지연)
Description : Event에 대한 자연어 설명
StartTime(UTC) : UTC 시간대 Event 시작 시간
EndTime(UTC) : UTC 시간대 Event 종료 시간 (영향이 끝난 시간)
Distance(mi) : Event에 영향 받는 도로 범위
AirportCode : Event 발생지에서 가장 가까운 거리의 공항 기반 기상 관측소
Side : 도로 상에서 Event의 위치 (L / R)
---------------------------------------------------------------------------------------------------------------------------------

SanFrancisco_traffic_Accidents.csv
---------------------------------------------------------------------------------------------------------------------------------
Severity : 사고의 심각도 0~4 / 0은 traffic에 미치는 영향이 가장 적음(짧은 지연) / 4는 트래픽에 상당한 영향(긴 지연)
StartTime :  사고 시작 시간
EndTime : 사고 종료 시간 (영향이 끝난 시간)
Description : 사고에 대한 자연어 설명
Start Lat,Lng : 영향 시작 좌표
End Lat, Lng : 영향 끝 좌표
Distance(mi) : 사고 영향받는 도로 범위의 길이
Description : 사고에 대한 자연어 설명
Side : 도로 상에서 Event의 위치 (L / R)
AirportCode : 사고 발생지에서 가장 가까운 거리의 공항 기반 기상 관측소
Temperature(F) : 온도
Humidity(%) : 습도
Visibility(mi) : 가시성 (운전자의 시야 범위)
Wind_Direction : 바람의 방향
Wind_Speed(mph) : 바람의 속도
Precipitation(in) : 강수량
Weather_Condition : 기상 상태
